# AAAK Template

Use this as a one-line note format for project memory.

## Basic Pattern

```text
PROJ: ABC | 2026-04-16 | *warm* what changed | why it matters | next step
```

## Examples

```text
PROJ: ADV | 2026-04-16 | *fierce* fixed timer double-count issue | render from saved seconds only | verify today log matches UI
```

```text
PROJ: ADV | 2026-04-16 | *bloom* added diary sync check | keeps history consistent across sessions | review latest entry before next change
```

```text
PROJ: ADV | 2026-04-16 | ⚠ avoid guessing facts | query palace first | prevents wrong context in replies
```

## Simple Rules

- Keep it to one line when possible.
- Put the date first.
- Use short action words.
- Add an emotion marker only when it helps.
- End with the next step or warning.
